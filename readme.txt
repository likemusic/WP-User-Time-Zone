== Description ==

**The "User Time Zone" plugin allows users to set their personal time zone on their profile page and view it on the frontend accordingly.**      
*Date and local time will be displayed on the frontend according to the personal time zone set-up by means of this plugin.*     
By default Wordpress doesn't have such ability. 

== Installation ==

1. Extract the zip file.
1. Upload the folder to the `/wp-content/plugins/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
